CSS: Hojas de Estilo

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación visual de documentos estrucuturados, habitualmente HTML. Su función principal es separar el contenido (marcado) de su apareiencia, permitiendo definir estilos para colores, tipografías, tamaños, márgenes, disposición en la página y efectos visuales. Al aplicar feglas de estilo a elementos concretos, CSS controla cómo se ve una interfaz sin alterar su esctructura semática. CSS funciona mediante reglas que asocian selectores (que identitifican elementos o grupos de elementos) con declaraciones de propiedades y valores. La cascada y la especificidad determina qué relgas prevalecen cuando vrias compiten sobre el mimsmo elemento, y las hojas de estilo puede organizarse en distitntos niveles (usuario, autor, navegador) para gestionar prioridades. Además incluye mecanismos para diseño responsivo, como unidades relativas, consultas de medios y sistemas de layout modernos, que facilitan adaptar la presentación a distintos tamañaos de pantalla y dispositivos. CSS también ofrece capacidades para animaciones, transiciones y transofrmaciones, permitiendo enriquecer la experiencia visual sin necesidad de scripts. Su modularidad y reutilización mediante veriablas y componentes favorecen matenibilidad y choerencia en proyectos grandes, consolodándolo como la herramienta esencial para el diseño visual en la web.
